# WebFlow AI - Premium AI-Powered Website Builder

[![Preview](https://img.shields.io/badge/Preview-Live-brightgreen)](https://website-builder-sixth.vercel.app/)

## 🚀 Features

- **Drag & Drop Builder**: Intuitive visual editor with real-time preview
- **10+ Pro Themes**: Minimalist, Corporate, E-commerce, Portfolio, Landing & more
- **200+ Components**: Hero, Features, Testimonials, Pricing, Forms, Galleries
- **Visual Effects**: Parallax, Scroll animations, Hover transforms, Particles
- **Responsive Design**: Perfect on Desktop, Tablet, Mobile
- **AI Compliance**: GDPR/CCPA ready with cookie banner & privacy tools
- **SEO Optimized**: Meta tags, Open Graph, Schema markup
- **One-Click Export**: Clean HTML/CSS/JS download
- **Sound Effects**: Satisfying UI interactions

## 🎯 Quick Start

```bash
npm install
npm run dev  # or npx serve .
```

Open [http://localhost:3000](http://localhost:3000)
